## Цель работы
Изучить основы контейнеризации с использованием Docker: установка, работа с образами, контейнерами, томами и создание собственного Dockerfile.

## Ход работы
Установила Docker и проверила его работу.
Запустила тестовый контейнер hello-world.
Изучила базовые команды (docker ps, docker images, docker run).
Запустила контейнер с образом ubuntu, установила внутри curl.
Развернула веб-сервер на nginx:alpine, открыла его в браузере.
Научилась работать с томами (volumes) и проверила сохранность данных.

## Результаты
Установлен и настроен Docker
Освоены базовые команды Docker
Получен опыт работы с образами Ubuntu и nginx
Реализовано собственное Flask-приложение
